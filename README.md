# bdsx-DropItemTag
This is a plugin that displays information about dropped items that can be used in BDSX as tags.

## Image


## Custom
You can customize the plugin by editing the [Setting](./setting.json) file.
```javascript
{
    "updateTick": 100,
    "tagText": "§r{name}§r §l§8| §fx{amount}"
}
```
